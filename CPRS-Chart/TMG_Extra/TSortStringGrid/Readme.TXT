To install the TSortStringGrid for Delphi:

- Click on Component and Install Packages.
- Click the Add... button, which will bring up the "Add Design Package" window.
- Navigate to the location of the unzipped TSortStringGrid folder.
- Select the dclTSortStringGrid100.bpl file and click Open
- Click OK on the Project Options window.

The TSortStringGrid should now be located in the Tool Palate, for use in design time forms.
